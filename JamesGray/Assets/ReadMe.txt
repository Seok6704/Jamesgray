Asset 폴더 구성 설명

1. Adaptive Performance

    Mobile로 프로젝트 생성시 자동으로 생성되는 폴더

2. Animations

    스프라이트 애니메이션 및 컨트롤러 폴더

3. Camera

    카메라 관련 스크립트

4. Editor

    자동으로 생성되는 폴더

5. Resources

    각종 Asset이 위치하는 폴더로 런타임 도중 스크립트에서 불러올 assets이 위치

6. Scenes

    Scene들이 위치하는 폴더

7. Scripts

    각종 스크립트들이 위치하는 폴더

8. Shader

    쉐이더 및 머티리얼이 위치하는 폴더

9. TextMeshPro

    TMPro 관련 데이터들이 위치하는 폴더

10. TutorialInfo

    자동으로 생성된 폴더
